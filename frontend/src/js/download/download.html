¿Quieres instalar facturascript en tu servidor y personalizarlo?

Como bingo es una aplicación web, necesita Apache, PHP y MySQL para poder ejecutarse. Siga los pasos para instalar XAMPP y después bingo.

1. Descargue e instale XAMPP

Instale XAMPP en Windows e inicie los servicios MySQL y Apache.

2. Descargue bingo

3. Descomprima y copie bingo

Descomprima bingo en C:/xampp/htdocs y renombre el directorio bingo a facturas, para mayor comodidad. A continuación acceda localhost/facturas para ir al instalador. No necesita cambiar ningún valor del instalador, simplemente pulse el botón aceptar.

4. Usuario y contraseña

Cuando le solicite usuario y contraseña escriba: admin como usuario y admin como contraseña.
